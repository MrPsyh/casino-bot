## 📝 Команды Пользователя

- `/start` - Начать
- `/profile` - Профиль
- `/balance` - Баланс

## 🔧 Команды Администратора

- `/admin_help` - Справка по командам
- `/add_admin <user_id>` - Добавить админа
- `/remove_admin <user_id>` - Удалить админа
- `/admin_list` - Список админов
- `/approve_withdrawal <код>` - Одобрить вывод
- `/reject_withdrawal <код>` - Отклонить вывод
- `/pending_withdrawals` - Ожидающие выводы
- `/create_promo <код> <награда> <лимит>` - Создать промокод
- `/list_promos` - Список промокодов

## 📊 Структура Базы Данных

- `users` - Пользователи и их статистика
- `transactions` - История транзакций
- `withdrawals` - Заявки на вывод
- `admins` - Администраторы
- `promo_codes` - Промокоды
- `promo_uses` - История использования промокодов

## 💳 Система Выводов

1. Пользователь нажимает "Вывести"
2. Выбирает сумму или вводит кастомную
3. Получает **уникальный код вывода** (например: #1, #a1, #b1 и т.д.)
4. Отправляет код администратору
5. Админ проверяет код и одобряет/отклоняет
6. При одобрении - отправляет звёзды пользователю
7. При отклонении - звёзды возвращаются на баланс
